┌─────────────────────────────────────────────────────────────────────────────┐
│ 🔧 TROUBLESHOOTING                                                          │
└─────────────────────────────────────────────────────────────────────────────┘

Demo videos not showing?
  → Check: ls datasets/raw/
  → Supported: .mp4, .avi, .mov, .mkv, .flv
  → Restart: docker-compose down && ./Start.sh

Analysis stuck?
  → Click "Stop" button or
  → curl -X POST http://localhost:8000/api/v1/videos/{id}/stop

Production camera not working?
  → Test RTSP:  ffprobe rtsp://your_url
  → Verify:     curl http://localhost:8000/api/v1/health
  → Check:      docker-compose logs backend

Both interfering?
  → They shouldn't! Check logs for errors
  → docker-compose logs | grep error
